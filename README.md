# keiba-research — verification-asset hash registry

日本競馬の市場効率性に関する独立研究プログラムの、検証資産
（事前登録・実行メタデータ・凍結集計・章本文・論文草稿）の
**sha256ハッシュのみ**を時系列で公開するレジストリです。

- 目的: 研究成果の公開タイムスタンプ（優先権）の確保と、
  「結果を見る前に判定基準を固定した」ことの第三者検証可能性。
- 中身（データ・本文）はここには置きません。ハッシュと突合可能な
  原本は、対応するプレプリント公開時に照合できます。
- `HASHES.md` は **append-only** です。行の削除・書き換えは行いません。
